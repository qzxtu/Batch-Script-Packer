# دفعة سيناريو باكر

تتضمن العملية قراءة محتويات ملف cmd / bat واستخدامها لإنشاء مجموعة وحدة تحكم C # ، والتي تعد في الأساس تطبيقًا مترجمًا. سيقوم هذا التطبيق بعد ذلك بتنفيذ الأوامر المحددة في الملف باستخدام تنفيذ shell.
