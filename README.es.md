# Empaquetador de secuencias de comandos por lotes

El proceso implica leer el contenido de un archivo cmd/bat y usarlo para crear un ensamblaje de consola C#, que es esencialmente una aplicación compilada. Esta aplicación luego ejecutará los comandos especificados en el archivo usando la ejecución de shell.
