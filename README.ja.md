# バッチ スクリプト パッカー

このプロセスでは、cmd/bat ファイルの内容を読み取り、それらを使用して C# コンソール アセンブリを作成します。これは、基本的にコンパイル済みアプリケーションです。このアプリケーションは、シェル実行を使用して、ファイルで指定されたコマンドを実行します。
